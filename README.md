## 🚀 Overview
This project is a web-based recreation and redesign of the **Windows 98 operating system**. It’s not just a UI clone; it's a functional simulation designed to showcase the power of modern web technologies in mimicking classic desktop environments.

### ✨ Key Features
- **📟 Classic UI:** Faithfully recreated taskbar, start menu, and icons.
- **📂 Window Management:** Draggable, resizable, and minimizable windows.
- **🛠 Functional Apps:** Integrated tools like Notepad, Calculator, and a File Explorer.
- **🎨 Pixel Perfect:** Every border and shadow is designed to match the original 98' experience.
- **📱 Responsive-ish:** Optimized for desktop browsers to maintain the original aspect ratio.

To achieve this level of simulation, I used:

- ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black) **JavaScript** for window logic and system functionality.
